基础心理课第二章 条件反射（一二）

第一节 条件反射

无条件反射：无条件刺激 ---> 无条件反应（先天固有）
条件反射： 条件刺激 ---> 条件反应（后天学习）
小狗实验：狗看到食物就会流口水，这是无条件反射（先天）；每次在给狗拿食物前都摇一下铃铛，时间一久只摇铃铛也会让狗流口水。
条件反射形成过程：无关刺激（如铃声），与无条件刺激（如食物）在时间或空间上有过一次或多次结合，而且无关刺激先于无条件刺激出现或二者同时出现，无关刺激就会逐渐取代无条件刺激引起有机体（如实验中的小狗）的某种反应（如流口水)。
条件反射是一种具有普遍意义的大脑活动方式。学习是条件反射建立的截断，而记忆是条件反射的巩固过程。

引起条件反射的信号刺激，包括三类：

  1. 第一信号：即现实的刺激，如声、光、电、味等。对第一信号发生反应的皮质机能系统，叫做第一信号系统。比如在你面前拿出了一个梅子，你会流口水，这是第一信号。
  2. 第二信号：即抽象的刺激，如语言、文字等。对第二信号发生反应的皮质机能系统，叫做第二信号系统。在人面前写青梅两个字，会流口水。这是人特有的。
  3. 第三信号：即各种念想、思维、欲望、感觉、感情等主观因素。对第三信号发生反应的皮质机能系统，叫做第三信号系统。也是人类特有的。

在日常生活中，人的大脑系统经常受客观刺激的作用不停地运转，并产生各种念想，而这些念想又构成抽象的主观刺激，引起有机体产生心理反应和行为反应的系列反应。我们把来自于大脑又直接作用于大脑的各种念想、思维、欲望、情感等主观因素，统称为第三信号。
其实，口吃病和社恐所表现出的各种症状，就是一系列条件反射被触发的结果。

一、无条件反射

无论是在异性、领导、陌生人面前，还是有求于人的，或面试、相亲、公众讲话时，或在一些激动人心、令人愤慨的场合，出现恐惧、紧张或兴奋情绪都很正常。把这些情况下发生口吃的过程，称作“口吃的无条件反射”。在无条件反射下发生的口吃属于常态口吃，不需要治疗。

二、条件反射

当个体受到某种看似与口吃无关的刺激而发生口吃，我们称之为“口吃的条件反射”，简称为“口吃的反射”。这种口吃属于病态口吃，属于心理问题，需要治疗。病态口吃都是条件反射的结果。
例子是：小明找老板汇报工作，但是由于当时老板生气而发生常态口吃，被老板助手小李全程注意，以后碰到小李也会有害怕心理，形成病态口吃。

三、中性刺激

即无关刺激，它在口吃反射形成前，不会引起口吃的预期反应。它是条件刺激的“前身“，任何无关刺激都可能会变成条件刺激。比如给狗喂食前打铃，久而久之，打铃也会让狗流口水。

四、无条件刺激

口吃的无条件刺激，是指在口吃反射形成之前就能引起个体发生口吃的刺激物或压力，它也叫超限刺激。

五、无条件反应

无条件刺激引发的反应，叫无条件反应，也叫正常反应。

六、条件刺激

口吃的条件刺激是指那些可以引起个体产生口吃的预期反应的刺激或信号，条件刺激也叫条件压力。从理论上说，凡是与无条件刺激（或稳定的条件刺激）有过结合或有过关联的一切主客观刺激物，都可能成为条件刺激。比如在给狗喂食前给一稳定的刺激（摇铃/拍桌子/...），以后都能成为条件刺激（摇铃等）。
从理论上讲，无关刺激都有可能变为条件刺激。
条件刺激可分为：

  1. 孪生刺激：能同时引起同一种条件反射的条件刺激称为孪生刺激。孪生刺激的特点是，其中一个刺激单独出现都会引起个体产生预期反应，也就是条件反应。比如赵总找小明谈话时，抽的烟、桌上摆的物体日后都可能引起个体（小明）产生预期反应。
  2. 复合刺激：特点是某一个刺激单独出现，或者信号数目发生变化，或者信号的组合顺序有改变，就不容易引起预期反应。比如有一辆卖东西的车每天下午5点会进村卖东西，到达时会按一下喇叭。你在家看电视时听到汽车喇叭声并知道现在是5点，那么你就会飞奔去买东西；但是某天你听到汽车喇叭声，但是你发现现在才4点钟，那么你会对此半信半疑，慢慢走出去看是不是卖东西的车来了。

七、条件反应

条件刺激通过条件反射引起的反应，叫条件反应。

八、反射中枢

无条件反射在大脑皮层下面有一个与之对应的神经中枢，我们称之为无条件反射中枢或者叫反射的先天种子。
条件反射形成后大脑皮层会自动形成一个与该条件反射匹配的、新的神经中枢，我们称之为反射中枢或者称为反射的后天种子。

九、多级反射

当一级条件反射稳定后，再用新的刺激与这一条件反射结合，还可以建立二级、三及甚至更多级的条件反射。

十、复合反射

当一系列首尾相连的复合刺激引起的反射结合在一起时，就会复合成全新的反射，我们称之为复合反射。比如：
反射F1: A引起B
反射F2: B引起C
反射F3: C引起D
那么上面3个反射可以复合成条件反射F4: A到D

第二节 条件反射的特征

一、获得

二、强化
某一条件刺激引起条件反射，出现条件反应后再呈现条件刺激，这个过程就称为条件反射的强化。

三、清退
如果条件刺激引起条件反射并出现条件反应后，不再呈现无条件刺激，即不强化它，那么已建立的条件反射就会逐渐消失。比如摇铃后不给狗喂食，那么狗就会说一句你大爷的，骗我，第二天如此，狗就会想你还骗我，第n天亦如此，重复多次后摇铃就不会让狗过来。条件反射能消退是口吃病能治好的根本保证

四、泛化
泛化是指条件反射建立后，与条件刺激类似的刺激也能激发相同的条件反应。比如一朝被蛇咬、十年怕井绳。

第三节 操作性条件反射

有机体知道行为与后效的关系后，依据后效来调节行为。例子：国家制定法律就是为了给公民制定规章制度，公民根据此来调节自己的行为。小孩偷鸡蛋，母亲就夸他并把蛋炒来吃，那么小孩就会继续偷，长大后就偷金。

第四节 反射机制

口吃反射是一个复合反射系统，其核心是显意识层建立了具有记忆和反射功能的神经中枢。

一、无条件反射f: A -> C + D + E
无条件反射f（A）引起反应C、D、E。A属于过强压力/超限刺激，C/D/E都属于无条件反应。任何人在超限刺激（A）下都会无条件的产生恐惧心理（C），以及与之匹配的生理反应（D）和行为反应（E）。

二、条件反射g: B -> C
口吃患者遇到条件刺激（B）后，会出现如下心理反应（C）：

  1. 想起丢人现眼的口吃经历而感到害怕
  2. 害怕再次发生口吃
  3. 闪现”我会口吃吗？“的念头
  4. 出现”我肯定会口吃“的念头
  5. 暗示自己”千万不能口吃啊“

三、无条件反射h: C -> D
心理反应（C）产生后，必然会导致生理反应（D）：心跳加快、呼吸紊乱、肌肉紧张、四肢发抖、声带闭锁、口腔痉挛、难发的感觉。

四、无条件反射i: D -> E
条件刺激（B）引起心理反应（C），心理反应（C）引发生理反应（D）之后，言语行为就会受阻，也就是难发，此刻患者们通常会作出以下行为反应（E）：

  1. 为冲破难发阻力用力去说，结果导致口吃，并引起各种奇怪的表情、手势以及动作。
  2. 为避免继续说下去可能会发生的口吃，选择了逃避，如换词、绕过去、故意停顿、还会百般掩饰逃避行为。
  3. 因为用力，说话急、快、强、重、乱，这时容易导致说话不清晰，说话有失控感。
  4. 如果用力过猛，有可能会导致强直性（像抽搐一样）口吃。

五、条件反射j: B -> E
g: B -> C
h: C -> D
i: D -> E
j: B -> E（绕过心理反应（C）和生理反应（D）直接出现行为反应（E），即复合反射），这意味着患者们在不紧张的情况下也可能发生口吃
由此可见，心理反应（C）是口吃反射的纽带和核心。

第五节 口吃信号

条件刺激（B）-> 心理反应（C）-> 生理反应（D）-> 行为反应（E）
B/C/D都能引发行为反应（E）

引发口吃的信号刺激有以下几大类：

  1. 发生口吃的人物、时间、地点、事物和环境、口吃的字等。
  2. 发生口吃时的各种生理反应，如紧张、收缩、心跳加快、呼吸紊乱、四肢发抖等。
  3. 口吃本身，以及身体伴随动作等。
  4. 第三信号，包括患者当时口吃的想法、念头、暗示、对抗等各种心理活动。
