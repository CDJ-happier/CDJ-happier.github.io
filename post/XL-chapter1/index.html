基础心理课第一章 心理阴影

一、心理阴影的由来/形成

心理阴影的形成主要有两个原因：

  1. 一是过往因为口吃而承受的压力
  2. 二是过后的耿耿于怀

口吃患者承受的压力有哪些？

每个口吃患者都承受着内外双重压力，即主观压力和客观压力，总共8大压力

客观压力

  1. 发生口吃受到嘲笑，而感到丢人现眼带来的压力。
  2. 受到父母或他人的不良暗示，如责骂、提醒、模仿等带来的压力。比如当你口吃时，父母或他人叫你慢点说话。
  3. 受到社会歧视、打击而形成的压力。
  4. 不良的人际关系带来的压力。意思是害怕和别人交流、害怕社交，心理扭曲。

主观压力

  1. 有口难言导致的压力。明明自己喜欢说话，有很好的想法想分享，可是因为害怕而不敢说话就很压抑。
  2. 屡战屡败导致的压力。结巴一次就很痛苦，我们的目标是不口吃，但是结巴一次就使愿望落空，结巴很多次就挫败很多次，很不痛快。
  3. 心理纠缠导致的压力。心理纠缠是最累的，说话前有口吃预感、口吃意识，自己就心想待会不要口吃、不要急。最典型的例子就是老师让你明天或下周登台演讲等，那么这段时间你就会一直处于忧虑之中，寝食难安。
  4. 逃避现实带来的压力。不敢与人交往，能逃避就逃避，能少说就少说。但这不是内向，其实内心是想去的。

如果一个事发生过后让自己不舒服就叫做创伤性经历，如果让自己舒服那么就是美好的回忆。创伤性经历每想一次就不舒服一次，如果老想的话就会成为心理阴影，心理阴影是一种记忆。一个事发生过后我不去想它，那么就过去了，慢慢会淡忘，但是每想一次就会强化一次记忆，想多了就会成为阴影，成为一种记忆。

客观压力具有一过性、主观压力则具有永久性和毁灭性

st=>start: 八大压力
cond1=>condition: 让你不舒服？
badres1=>inputoutput: 创伤性经历
goodres=>inputoutput: 无影响/逐渐忘掉
cond2=>condition: 事后耿耿于怀、念念不忘？
badres2=>inputoutput: 心理阴影
e=>end: end

st->cond1
cond1(yes)->badres1
cond1(no)->goodres
badres1->cond2(yes)->badres2
badres1->cond2(no)->goodres

如果发生的某件事让你不舒服，那么就相当于发生了创伤性经历；如果你不认为这是一件让你不舒服的事，那么就没有什么问题。

  1. 客观压力 = 创伤性经历(如果自己事后耿耿于怀，那么就会变成心理阴影）
  2. 创伤性经历 + 耿耿于怀、念念不忘 = 心理阴影

口吃患者对口吃以及外在压力的态度

正常人也有很多压力，但并不是所有的压力都会成为心理阴影（潘石屹上央视也结巴，但是他却没有看得多重，没有多在意）。
虽然压力无处不在，但并非人人都会落下心理阴影。事物的发展离不开内因和外因两个因素，其中内因是根本，外因是条件，外因通过内因起作用。压力能否造成心理阴影，个人的认知态度往往起到决定性作用。即心理创伤是心理阴影的雏形，八大压力造成心里创伤。

心理创伤 + 正确的认知态度（不耿耿于怀、不纠结） = 心理创伤化解、慢慢淡忘
心理创伤 + 错误的认知态度（耿耿于怀、念念不忘） = 就会形成心理阴影

可见：
八大压力 + 错误的认知态度（耿耿于怀、念念不忘） = 导致心理阴影
但是从整个社会看来，口吃确实会被歧视，而且很难改变这个现象，所以我们需要改变自己、改变我们的认知态度。态度源于认知，认知决定态度，认知和态度改变后我们就基本彻底好了。

二、心理阴影的危害

常人也有口吃、难发，但没有口吃预感，因为口吃预感来源于心理阴影。

心理阴影形成后，一旦遇到合适的刺激，就会以条件反射的形式发作，使你产生口吃预感、口吃意识、恐惧感、甚至发生口吃。
其实治口吃病就是一个淡化心理阴影的过程。前面说过八大压力 + 错误的认知 = 心理阴影
因此如何淡化心理阴影呢？：

  1. 先止损，即停止制造新的心理阴影。应尽力避免八大压力，其次改变错误的认知态度

  2. 淡化已经产生的心理阴影
